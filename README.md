# Yaraav Prompt - Smart Prompt Manager

Yaraav Prompt is a sleek, modern, and highly functional desktop application designed for AI enthusiasts, developers, and prompt engineers to seamlessly organize, manage, and optimize their AI prompts. Inspired by the premium and minimalistic Twitter/X dark UI, Yaraav Prompt provides a fully local and secure environment to store your creative assets.

## ✨ Key Features

* **𝕏-Inspired Dark UI:** A gorgeous, high-contrast dark theme (#000000 deep blacks) engineered for long coding and writing sessions.
* **📂 Project & Category Management:** Create dedicated projects and categories to group your prompts logically and filter through them instantly.
* **⚡ Smart Auto-RTL Detection:** Advanced support for Persian and Arabic text. The application automatically detects the language and shifts the alignment to Right-to-Left (RTL), with manual LTR/RTL override switches on every card.
* **📋 One-Click Quick Copy:** A seamless copy button on each prompt card that saves text instantly to your clipboard with smooth visual feedback.
* **🔍 Dynamic Live Search:** Instantly query and filter through your titles, content, projects, or categories as you type.
* **📊 Character Counter:** Live tracking of character lengths on each prompt card to help you monitor token boundaries.
* **🔒 100% Offline & Private:** Built with standard web technologies utilizing browser `LocalStorage`. Your data never leaves your local machine—zero external databases, zero tracking.

## 🛠️ Tech Stack

* **Frontend:** HTML5, Tailwind CSS (via CDN), Vanilla JavaScript
* **Desktop Wrapper:** Nativefier / Electron (Compiled for Windows Desktop)

## 🚀 How to Run Locally

If you wish to run or modify the source code directly:
1. Clone this repository or download the `index.html` file.
2. Double-click `index.html` to launch it instantly in any modern web browser.

## 📦 Download Windows App (.exe)

To run Yaraav Prompt as a native Windows desktop application without opening a browser:
1. Navigate to the **Releases** section on the right side of this repository.
2. Download the latest `Yaraav-Prompt-Windows.zip` archive.
3. Extract the ZIP file completely.
4. Launch `Yaraav Prompt.exe` and enjoy!
